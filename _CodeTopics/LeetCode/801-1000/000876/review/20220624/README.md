# - 奇数个节点时，最终慢指针停留在中间那个节点，快指针停留在 None；

#

#
#               slow           fast

# 此外，这两种情况的差异还有如下表现：
## 情况1. 
## 从人的角度去数数（也就是下标从 1 开始算）
## slow 处于第 x 个节点时（也就是走了 x 步），fast 一定处于第 2x 个节点；
## 但是实际上 slow 的节点 index 是 x-1，fast 是 2x - 1。

# 情况2. 
## 从机器的角度去数数（也就是下标从 0 开始算）
## slow 处于第 x + 1 个节点时（也就是走了 x 步），fast 一定处于第 2x + 1 个节点；
## 但是实际上 slow 的节点 index 是 x，fast 是 2x 。
