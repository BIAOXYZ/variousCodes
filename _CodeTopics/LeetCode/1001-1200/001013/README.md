# `WA--001013.py` 和 `001013.py`

`WA--001013.py`对于输入`[1,-1,1,-1]`是无法得到正确答案的。`001013.py`做了一些改进：第一轮while循环直接从第二个数开始，第二轮while循环也是直接从新的下标的下一个开始。这样保证每次求和都至少加了一个数组A中的元素。
