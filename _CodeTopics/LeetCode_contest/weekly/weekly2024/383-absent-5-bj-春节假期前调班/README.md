    + 出发前发现打车太堵了，就决定试试地铁（`五棵松站`--`大望路站`），然后发现 SKP 竟然也直通地铁了！！！真心牛，以前还半开玩笑地说 SKP 作为顶级商业中心，只是门口有地铁，虽然很近，但毕竟地下没有直通地铁，结果没想到它还真搞定了（***并且明显是强行开了一段不算短的距离，然后跟地铁连上了***）。能量不小啊- -
