python的按位亦或运算符是`^`
> Python 运算符 https://www.runoob.com/python/python-operators.html
```py
a = 60            # 60 = 0011 1100 
b = 13            # 13 = 0000 1101
c = a ^ b;        # 49 = 0011 0001
print "c 的值为：", c
--------------------------------------------------
c 的值为： 49
```
