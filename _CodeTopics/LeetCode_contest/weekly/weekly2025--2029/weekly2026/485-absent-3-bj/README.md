
# 缺席

- 本周一（`20260112`） 
  * 晚上媳妇儿回来，发现喉宝拉到阳台上了，这是之前从来没有过的情况。虽然便便稍有点稀，但当时觉得可能就是一时不舒服，甚至是对我们带她去绝育的报复，就没太上心。
- 本周二（`20260113`） 
  * 今天带喉宝去了两趟宠物医院：第一趟是按照之前做绝育手术完医生的嘱咐，今天去医院检查下手术后的伤口愈合情况，属于计划中的；第二趟完全是计划外的，是因为她不舒服（大概率是肠胃炎）。
    + 昨天特意申请了今天一天在家办公，准备中午媳妇儿回来一起带她去医院看伤口。结果媳妇儿发现`喉宝`吐在了阳台那个废弃的猫抓板上了，当时也没有引起注意。然后去医院医生说伤口恢复挺好的，于是就回来了。
    + 然后下午17点多的时候突然发现被子上一滩类似水的东西，当时以为是她尿被子上了，后来发现是吐的。然后又发现阳台地上还有两滩，并且这两滩是浅粉色的。当时吓坏了，以为是吐了些血。赶紧告诉媳妇儿，她直接从单位打了个车，途径点设置到楼下，然后一起又到医院去了。还好就是`喉宝`精神一直不错，医生看了后也觉得应该还好，最后是打了止吐针，输了一些液，然后又回来了。为以防外一，还在宠物医院买了一盒`泰淘气益生菌`，小计 **`100元`**——知道网上肯定便宜，但是第一检查没收费，觉得还是让医院赚点；第二是自己买确实也慢。
- 本周四（`20260115`） 
  * 这两天`喉宝`状态都很一般，于是我们除了罐头里拌益生菌外（但她吃得很少），媳妇儿还在网上买了宠物用的`阿莫西林克拉维酸钾`和注射器。第一次药水混着猫条一起，没费劲就成功喂下去了；第二次才用上注射器（此时她已经好了很多，反抗力度不小，反而没喂多少，主要是靠第一次喂的）。
  * 今天下班回家，发现`喉宝`又吐了一次，这次有点偏绿色。但是好消息是她看起来精神还不错，总体还是在慢慢好转的。
- 本周六（`20260117`） 
  * 今天早上起来看东西，`喉宝`一直捣乱（感觉已经好很多很多了），直接卧到笔记本上。。。让她卧了一会儿后，给她拉开沙发套，果然就钻进去睡觉了。
  * 这两天她肠胃病情况在好转，尤其是今天晚上，感觉已经完全恢复了，精神特别好，像以前一样很皮。就是食欲跟以前比起来还是差一点，但是（疑似的肠胃病）应该是基本好了。——不禁感叹以为绝育前做足了工作，没想到还有遗漏的点：冬天太冷容易肠胃炎。
- 本周日（`20260118`） 
  * 今天去医院给`喉宝`拆线，这个完全是小手术，甚至都不能称之为手术——两三分钟就搞定了。然后医生说两天后就可以不用穿绝育服了！
  * 回来后看她精神状态很不错，也吃了不少东西，于是俩人想着可以出去了，就去了中关村大融城。
    + 先是去那边的`胖妹面庄 (中关村大融城店)`吃了个饭，杂酱面我感觉一般，面太少，菜太多；媳妇儿叫的酸辣粉说是还行。
    + 然后去这里的`TOP TOY(北京中关村大融城店)`给媳妇儿买了一个怪兽电力公司的盲盒。
    + 接着是在这里的`三月兽(中关村大融城店)`给媳妇儿买了一个小埋的盲盒——这家店的二次元味道比较浓厚，甚至有葬送的芙莉莲的盲盒——真的还挺不错的。
    + 还去了这边迪卡侬对面的宠物店（好像是叫`美天灵宠物`），结果发现价格非常虚高，就是坑毛孩子主人。于是什么也没买就出来了。
  * 后来又去领展一层的`贝果博物馆(中关村店)`小转了一下，给媳妇儿买了两个贝果。现做的那个牛肉的很不错，但蒜香的比较一般。不过对这家店整体感觉不错，很有朝阳区的风格。
