- 本周六（`20260124`）

- 本周日（`20260125`） 
  * 今天先是十点多起来去扔了下垃圾（很大的一个箱子以及`喉宝`用坏的猫抓板），然后在楼下每周二、四、日出摊的蔬果车那里买了些水果。
  * 等中午媳妇儿加班回来，就一起打车去了王府井。先是在`王府井百货大楼`的老铺黄金排队，排了二十多分钟进去了，但是唯一看得上的一个戒指只剩展览款了，最后想了想还是不买了。
  * 然后去`王府中环`逛了下，这里的老铺也是排长队。。。所以根本也就没去排了。此外还看了看之前在国贸没看成的`山下有松`，但是没有太心动要买的——有一款也是看着还行，但是没法完全满意。
  * 接着是`东方广场`和`喜悦`，都基本没逛什么就出来了。东方广场的老铺是总店，排队的更多了——也不知道今天怎么了，王府井商圈三家都这么多人——虽然有满1000减100的活动，但是我感觉还是因为知名度上来了。。。
  * 接着是去APM吃了个饭，好容易到了有松鹤楼的地方，吃的当然是松鹤楼（`松鹤楼面馆(北京apm店)`）。
  * 接着折回王府井百货大楼，去刚来的时候发现的一家面包店（`Rose&Cabin露丝卡文·世界冠军面包(王府井百货大楼店)`）给媳妇儿买了几个面包。后来证明这家店确实可以，尤其是蒜香面包，媳妇儿很喜欢。那以后只要来王府井的话，都可以顺便买点。
  * 最后是去`王府半岛酒店`一层的LV和香奈儿逛了下。LV的包没啥喜欢的，香奈儿倒是有一个我觉得还行，但是媳妇儿觉得还是一般，最终还是没买。
