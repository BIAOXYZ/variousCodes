
# 缺席

**假期调休调班**：整个春节法定假期共 **`9`** 天（***`20260215` 至 `20260223`***），史上最长。我年后还请了三天年假，等于休了 **`12`** 天。

本周的七天全部是法定假期，但之所以目录标题名字是`春节假期-II`，是因为实际上上周日（`20260215`）已经开始春节假期了，只是上周如果又记录`春节假期前调班`，又记录`春节假期-I`，名字就太长了。

- 本周一（`20260216`） `腊月二十九`（***`除夕`，正式放假第二天***）。
  * 今年和[去年](../../weekly2025/435-absent-5-bj-xa-sy-jz-bj-春节假期-I/README.md)一样，***也是没有农历的`大年三十儿`，所以`腊月二十九`就是`除夕`了***。
  * 睡到中午起来，吃了米饭 + 腊肠。然后因为老的台式机不知道为什么 Dota2 只能用默认的 AI 了，所以沮丧了好一阵子。晚上我是先把中午的米饭吃完，又吃了一些饺子。然后和媳妇儿、爸妈四个人看了春晚——春晚像往常一样无聊至极。
- 本周二（`20260217`） `大年初一`（***正式放假第三天***）。 
  * 上午十点多起来，然后和大伯一家吃了个饭，地点就在我家南边的一个饭店。这次预定非常难，也让媳妇儿见识到了一点点河南震撼：
    + 大厅桌子满到密密麻麻，比婚礼宴会的桌子密度高多了。基本上桌与桌之间也就是走个人的宽度- -。
    + **`598元`** 一桌的套餐，菜得有20多个（我们吃饭的人才7个- -）。早就吃饱了，后面还上了大肘子、大鲈鱼之类的硬菜（并且前面已经一堆硬菜了）。。。
  * 下午去见了一个过去多年的老同学，这哥们也是沉沉浮浮，最近算是勉强稳定下来了。
    + 本来没想着去他家，但是要开车去找网吧，所以去了下。给他儿子刷支付宝发了 **`500元`** 的压岁钱红包。
    + 后来开车去找网吧，没想到整个zz区竟然就一家网吧了，最后是跑到旁边的区才找到。上网也贼麻烦，要下一个专门的app认证（即使带了身份证也不行）。随便上去，申请了个kk对战平台的账号，玩了两局 dota1 就回去了。
- 本周三（`20260218`） `大年初二`（***正式放假第四天；去西安***）。
  * 初一按习俗不让洗澡，所以初二早上起来洗了个澡。今天按过去习惯该去大舅家，但是因为饭店没订到，改到初三了。今天下午就要坐高铁去岳父岳母家了，所以这次就不用去了。
  * ***下午就坐从`郑州东`中转的高铁，晚上18点多到了`西安北`***。岳父开车来接，回去吃了点儿饭就睡了。
  * 此外，今天在高铁上的时候，媳妇儿同事帮忙去家里看了看`喉宝`。主要是帮忙开小米的摄像头（这次走的时候忘了，把小米摄像头的插座关了- -）；又喂了她一根猫条和给她铲了铲屎。
- 本周四（`20260219`） `大年初三`（***正式放假第五天***）。 
  * 今天就是去媳妇儿舅舅的医院，看了看媳妇儿外公外婆，然后吃了个饭。
  * 下午媳妇儿睡了会儿，我则是看电脑，干活（人才计划申请书 + 写书）和上网。
  * 晚上和媳妇儿，岳父岳母一起去他们那儿的一个小景点（`清河`）转了转。
- 本周五（`20260220`） `大年初四`（***正式放假第六天***）。  
  * 今天是比较轻松的一天，在家里哪儿也没去，上午睡到了十一点多，下午媳妇儿看电视，我继续昨天的开电脑干活和上网。
  * 晚上四个人玩了一会掼蛋；今年是我和媳妇儿一队，岳父岳母一队，结果我们两局都输了。
- 本周六（`20260221`） `大年初五`（***正式放假第七天；回京***）。
  * 今天早上起来，一家四口去`浐灞`的`御锦城`，去媳妇儿姑姑家看她爷爷奶奶。感觉二老住得还挺开心的。
  * 然后坐到下午两点左右，就出发去高铁站，***坐下午三点多`西安北`到`北京西`的高铁返京***。
  * 到家后把`喉宝`开心的啊。但是很快发现`喉宝`声音有点异常，尤其是呼噜噜的时候，跟卡了痰似的。娃也似乎一直在跟我们说她不舒服，一直（带着那种异常的声音）呼噜噜的。到夜里三点多，我去睡了；媳妇儿醒过来继续观察她的情况。
- 本周日（`20260222`） `大年初六`（***正式放假第八天；我加班一天***）。
  * 今天我是春节期间加班一天，早上起来直接去单位加班了。媳妇儿带`喉宝`去宠物医院看了下，结果是有非常轻微的炎症。医生说不用药也可以，但是用药了就得连着5天天天吃。
  * 最后我们俩决定还是不用药了，但是放弃之前规划的上海之行，改为在家陪她。这家伙似乎能知道这些，声音一下子就好了很多很多。
