
# 缺席

本周媳妇儿应该算是（单位正式开始施行的）单休的第58周（春节期间有3周不好计算，所以忽略了）。

- 本周一（`20251208`） 
  * 新疆出差，去程。
- 本周四（`20251211`） 
  * 新疆出差，回程。
  * 又TM延误了，乌鲁木齐一到冬天下雪太多了，各种延误。这次我和一个同事没买到周四上午 `10:25` 的那趟，然后领导又想大家一起去机场。于是先定了下午16点多的国航的一趟，然后又换成13点多海航的，换来换去烦死了。。。
  * 还好海航的机票送了自营贵宾厅`海翼轩`。但是早上过去的时候还没开门。。。后来打电话说是第一趟飞机起飞前一个半小时才开门- -不过 `11:30` 时候也开门了。里面六个热菜，可以点三四种面和饭，其他零食饮料就不说了。总体上中规中矩吧。
  * 结果还是晚点了一个小时左右起飞，到达时间也晚了更多。海航在首都机场是在 T2 的——也是很久没来过 T2 了。这次是第一次用汇丰的接送机服务，感觉还行。因为前面说的换机票的经历，接机时间肯定也得改。基本上就是打了个电话就解决了，所以觉得汇丰的这个服务还不错。
- 本周六（`20251213`） 
  * 今天整整干了一天，结合着AI，改了一些代码并且搞了50个测试用例，把三个结题用的文档算是搞完了——没AI的话，光这50个测试用例都写不完。
- 本周日（`20251214`） 
  * 今天中午起来，和媳妇儿一起打扫了下家里的卫生，然后决定下午去刚开没多久的`中关村大融城`转转。
  * **`Shake Shack北京大融城店`**：到了就发现 Shake Shack 在这边也开了家店，于是果断给媳妇买了个汉堡 + 薯条 + 可乐的套餐。因为是新店开业，还送了个玻璃杯，不错不错。另外这家店的桌子上安装的是无线充电，有点小牛啊。
  * **`番茄口袋（北京中关村大融城店）`**：一家精品店吧，人不少（今天大融城整体人都不少）。在里面逛了逛之后给媳妇儿买了顶帽子——结果戴了没两天就开线了，质量不行啊。
  * **`OCE（北京海淀中关村大融城店）`**：门口宣传说实付满`69元`，送哈基米帆布袋，于是肯定要去买了。但是没想到这家不但东西不错，价格还贼便宜（就是店的布局过于标新立异，人在里面不太好走。。。）。凑了半天，才算买了70多的东西，选了个蓝色的四只猫猫帆布袋。
  * **`张拉拉兰州手撕牛肉面（大融城店）`**：这家超出了预期：面应该是手工拉面，很筋道；一碗面面量不算小了，而且可以免费续面；面的味道也不错；还有免费小菜小零食等。这个后续肯定是还可以来的。
  * **`好利来（新中关店）`**：最后是去新中关的好利来（大融城也有一个，但是当时已经比较想回了，没去细找）买了个它们的新品乳酪——果然没有小红书上说得那么好——味道就正常水平，个头又偏小，所以不是很值。
