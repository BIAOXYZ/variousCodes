  * 上午三件事：
    + 1.去买了礼炮、部分胸前戴的花、婚车的拉花
    + 2.我去理了个发
    + 3.去多氟多酒店看了看房子，发现那个酒店不小，设施啥的也还行吧。
  * 下午陪媳妇儿去婚纱那里试装（我没全程陪同，下面会说），我就知道她试了敬酒服，还有中式的秀禾。
